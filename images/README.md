# Avatar images

Place one image per avatar in this folder. When a respondent’s result is that avatar, the image is shown in the result section. If the file is missing, the Japanese kanji for that avatar is shown instead.

| Filename | Avatar | English name | Kanji |
|----------|--------|--------------|-------|
| `shokunin.png` | Shokunin | Devoted Craftsman | 職人 |
| `bushi.png` | Bushi | Disciplined Warrior | 武士 |
| `zenso.png` | Zensō | Reflective Sage | 禅僧 |
| `bosatsu.png` | Bosatsu | Servant of the Whole | 菩薩 |
| `gemba_no_hito.png` | Gemba no Hito | Source Rootician | 現場 |
| `entei.png` | Entei | Patient Gardener | 園丁 |
| `zenkai.png` | Zenkai | Complete Transformer | 全改 |

Recommended: square or near-square, at least 400×400 px. Formats: PNG, JPG, or WebP. Paths are relative to the survey root (e.g. `images/shokunin.png`).
